# Chocolate-sales.
This dashboard gives information about the chocolate sales across period, amount and countries. 
An interactive business intelligence dashboard analyzing chocolate product sales performance across countries, salespeople, months, and products. Built with **Power BI**.

## Key Insights

- **Total Sales Amount** 
- **Best Performing Country** 
- **Worst Performing Country** 
- **Top Salesperson** 
- **Lowest Performing Salesperson**
- **Best-Selling Product** 
- **Least-Selling Product** 

## Monthly Sales Trend

Area chart showing sales distribution across the year:

- Peak month: **May** (~$10k)
- Lowest months: **August–September** (~$1–2k)

## Dashboard Sections

1. **Overall KPI Cards** – Total sales, best/worst country & salesperson
2. **Monthly Sales Trend** – Area chart by month
3. **Sales by Country** – Horizontal bar chart
4. **Salesperson Ranking** – Combined column + line chart (ranking + boxes shipped)
5. **Product Performance** – Highlighting top and bottom products

## Technologies Used

- **Power BI Desktop**
- DAX measures & calculated columns
- Data source: CSV
